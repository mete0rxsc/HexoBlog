### Recent Projects  
Recently, I developed several Vue plugins compatible with the Yun theme.  

[valaxy-yun-external-link (GitHub)](https://github.com/mete0rxsc/valaxy-yun-external-link)  
**valaxy-yun-external-link** is a security validation Vue component that verifies whether user-input URLs are safe. It supports Valaxy's native Yun theme.  

[valaxy-yun-NoticeBar (GitHub)](https://github.com/mete0rxsc/valaxy-yun-NoticeBar)  
**valaxy-yun-NoticeBar** is a Vue component that provides a notice bar on the homepage, also supporting Valaxy's native Yun theme.  

I've published these projects on GitHub to make them available to more users (though this blog framework remains quite niche).
